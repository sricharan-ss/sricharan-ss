# 🚀 Featured Projects

## 🍽 FeedForward — AI Food Rescue Platform

AI-powered platform that connects **restaurants and donors with NGOs** to reduce food waste using **AI + geolocation**.

### Key Features

• Natural language NGO food requests
• AI converts request into structured filters
• Distance-based ranking of donations
• Real-time claiming workflow
• Firebase real-time data sync

### Tech Stack

Flutter • Firebase Auth • Cloud Firestore • Firebase Storage
Python Flask • Groq LLaMA AI • Geolocation APIs

# 🐍 Contribution Snake

Add this GitHub Action after creating the README to generate the snake animation.

```yaml
name: generate snake

on:
  schedule:
    - cron: "0 0 * * *"

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: sricharan-ss
          outputs: |
            dist/github-contribution-grid-snake.svg

      - uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
```

Then add this image to your README:

